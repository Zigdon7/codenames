# Codenames Remake
A modern TypeScript rebuild of Codenames with WebSocket multiplayer.

## Features
- Real-time multiplayer via WebSockets (Node.js)
- Vanilla DOM/CSS frontend powered by Vite
- Server-authoritative spymaster roles
- Lobby system with team and role selection

## Setup
1. `cd backend && npm install && npm start`
2. `cd frontend && npm install && npm run dev`
